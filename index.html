Проектирую чистую архитектуру баз данных SQLite3, бесшовно интегрирую сторонние API сервисы, обеспечиваю безопасность токенов и конфигураций через изолированные переменные окружения (`.env`) и настраиваю отказоустойчивость скриптов при высоких нагрузках.
