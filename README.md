## 🛠 Технологический стек сайта
* **Markup:** HTML5, CSS3 (Custom gradients & glassmorphic neon borders)
* **Styling Framework:** Tailwind CSS (via CDN)
* **Icon Pack:** FontAwesome v6.4
* **Engine / Скрипты:** Чистый JavaScript (Vanilla JS). Реализован динамический синглтон-движок для управления модальными окнами, что полностью предотвращает конфликты слоев (`z-index`), утечки памяти и блокирует фоновый скролл при просмотре кейса.

---

## 💼 Демонстрируемые кейсы (Python / Бэкенд)

В портфолио интегрированы ключевые проекты, разработанные для клиентов:
1. **Telegram Notes Bot** (aiogram, FSM-состояния, логирование)
2. **HR-платформа найма** (telebot, автоматизация рекрутинга, Double Opt-In логика)
3. **Бот с интеграцией SQLite3** (telebot, проектирование реляционных БД, модульные SQL-запросы, изоляция `.env` конфигураций)
4. **Информационный Погодный Бот** (Асинхронные HTTP-запросы, интеграция OpenWeatherMap API, обработка JSON)
5. **Конвертер Валют** (telebot, интеграция финансового API, Inline-клавиатуры)
6. **Бот расписания** (Автоматический расчет четности недель, Time-Management логика)

## 🎯 Особенности реализации интерфейса
* **Performance:** Ноль тяжелых JS-фреймворков. Сайт загружается мгновенно.
* **UX/UI:** Кастомные неоновые эффекты размытия (blur туманности), полностью адаптивная сетка (Grid) под мобильные устройства и планшеты.
* **Stability:** Код модальных окон оптимизирован. Вместо размножения скрытых DOM-элементов используется одно универсальное окно, наполняемое данными динамически из JSON-структуры в JS.
